# Team Management App

- This project has been developed to manage your team.
- User can login, logout, create new teams, and new players, and can delete them.
- Non-authorized users can not make changes on teams and their players.

# Technologies Used

- [x] ReactJS
- [x] REDUX
- [x] Bootsrap 5
- [x] NodeJS
- [x] MongoDB
- [x] Socketio
